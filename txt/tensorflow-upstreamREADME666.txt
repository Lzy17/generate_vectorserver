<h1>TfLite asynchronous execution</h1>
<p>WARNING: This feature is experimental and subject to change.</p>
<p>Experimental support for TFLite asynchronous execution and interoperability.</p>
<h2>Directory structure</h2>
<p>| Directory          | Description                                         |
| ------------------ | --------------------------------------------------- |
| <code>/async</code>           | Asynchronous execution APIs. Definition for async kernel.  |
| <code>/async/interop</code>   | Data structures supporting buffer and sync object interop. Reconciliation functions for buffer / sync attributes. |
| <code>/async/interop/c</code> | C APIs for buffer and sync object interop.          |</p>
