<h1>Distributed TensorFlow</h1>
<p>This directory contains the initial open-source implementation of the
distributed TensorFlow runtime, using <a href="http://grpc.io">gRPC</a> for inter-process
communication.</p>
<p>To learn how to use the distributed runtime to create a TensorFlow cluster,
see the <a href="https://www.tensorflow.org/deploy/distributed">Distributed TensorFlow</a> How-To.</p>
