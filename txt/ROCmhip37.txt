<h1>HIP</h1>
<p>HIP is both AMD's GPU programming language extension and the GPU runtime. This
page introduces the HIP runtime and other HIP libraries and tools.</p>
<h2>HIP Runtime</h2>
<p>:::::{grid} 1 1 2 2
:gutter: 1</p>
<p>:::{grid-item-card} {doc}<code>HIP Runtime &lt;hip:index&gt;</code>
The HIP Runtime is used to enable GPU acceleration for all HIP language based
products.</p>
