<h1>Custom Ops Examples</h1>
<p>The following subdirectories have examples of Custom Ops.</p>
<ul>
<li>multiplex_1: starting example, similar to np.where</li>
<li>multiplex_2: GPU (and CPU) example, similar to np.where</li>
<li>multiplex_3: dispatch to special case kernel (and sparse tensors)</li>
<li>multiplex_4: C++ backward compatibility example: lists of tensors</li>
<li>simple_hash_table: internal state using a Resource using ref-counting</li>
<li>sleep: Asynchronous (non-blocking) sleep op using AsyncOpKernel</li>
</ul>
