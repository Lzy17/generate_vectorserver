<h1>TensorFlow C++ Examples</h1>
<p>This directory contains examples of the TensorFlow C++ API (and some redirects).
If that's not what you're looking for here are some links:</p>
<ul>
<li>For TensorFlow python examples see
    <a href="https://tensorflow.org/tutorials">the tutorials on tensorflow.org</a></li>
<li>For community maintained keras examples goto
    <a href="https://keras.io/examples/">keras.io/examples</a></li>
<li>For TensorFlow Lite examples see
    <a href="https://github.com/tensorflow/examples/tree/master/lite">the tensorflow/examples repository</a></li>
<li>For the Udacity course notebooks, refer to
    <a href="https://github.com/tensorflow/examples/tree/master/courses">this directory</a></li>
</ul>
<h2>About these examples</h2>
<ul>
<li>The C++ API is only easily buildable from within the TensorFlow <code>bazel</code>
    build. If you need a stand alone build
    <a href="https://www.tensorflow.org/install/lang_c">see the C API</a>.</li>
<li>This directory is not actively maintained.</li>
</ul>
