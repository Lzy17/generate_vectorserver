<h1>TPU Driver API</h1>
<p>This repository contains the TPU driver API and network (gRPC) transport
implementation for high-performance access to TPU hardware.</p>
<h1>Building</h1>
<p>Bazel is used to build the driver library and tests. Remote tests will require
access to a Cloud TPU.</p>
<h2>Fetching Bazel</h2>
<p>Download the latest copy of Bazel from
https://github.com/bazelbuild/bazel/releases.</p>
<h2>Building</h2>
<p><code>bazel build ...</code></p>
<h2>Testing</h2>
<p><code>bazel test ...</code></p>
