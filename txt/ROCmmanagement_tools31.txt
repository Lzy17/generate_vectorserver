<p>:::{grid-item-card} AMD SMI
The AMD System Management Interface Library, or AMD SMI library, is a C library for Linux that provides a user space interface for applications to monitor and control AMD devices.</p>

<li><a href="https://github.com/RadeonOpenCompute/amdsmi">GitHub</a></li>
<li><a href="https://github.com/amd/go_amd_smi#example">Examples</a></li>
</ul>
<p>:::</p>
<p>:::{grid-item-card} <a href="https://rocmdocs.amd.com/projects/rocmsmi/en/latest/">ROCm SMI</a>
This tool acts as a command line interface for manipulating and monitoring the AMD GPU kernel, and is intended to replace and deprecate the existing <code>rocm_smi.py</code> CLI tool. It uses <code>ctypes</code> to call the <code>rocm_smi_lib</code> API.</p>
<ul>
<li><a href="https://rocmdocs.amd.com/projects/rocmsmi/en/latest/">Documentation</a></li>
<li><a href="https://github.com/RadeonOpenCompute/rocm_smi_lib">GitHub</a></li>
<li><a href="https://github.com/RadeonOpenCompute/rocm_smi_lib/tree/master/python_smi_tools">Examples</a></li>
</ul>
